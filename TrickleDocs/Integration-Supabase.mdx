---
title: 'How to Integrate with Supabase'
description: 'Learn how to connect your project to Supabase.'
---

Follow these steps to integrate your app with Supabase.

1. Register a Supabase account.
2. Create a new Supabase project.
3. Get your project's API URL and anon key from **Project Settings** > **API**.
4. Install the Supabase JavaScript client:
   ```bash
   npm install @supabase/supabase-js
   ```
5. Initialize the client in your application and start making requests.
