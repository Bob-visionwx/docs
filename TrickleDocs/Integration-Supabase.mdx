1. Register a Supabase account.
2. Create a new Supabase project.
![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/06fc6059-5ec4-43ed-bee7-1d888c1ffcc5/fc548f6c-ccf3-4ae1-aefa-4a85e545390e/image.png)
    
3. Connect your Supabase project to your Trickle project using knowledge.
    
    3.1.   Copy your supabase project url and anon public api key, they will be used in the 3.4. step.
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/06fc6059-5ec4-43ed-bee7-1d888c1ffcc5/938930d8-3a8c-420d-be52-83b1934a9f77/image.png)
    
    3.2.   Copy your Trickle project site url, they will be used in the 3.4. step.
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/06fc6059-5ec4-43ed-bee7-1d888c1ffcc5/6467bf86-9787-45e8-b88b-4efc73f5e540/image.png)
    
    3.3.    Configure the `Trickle project site url` in your Supabase project. By doing this, your Supabase project will redirect to this site url when your app user finish account sign up and email verification.
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/06fc6059-5ec4-43ed-bee7-1d888c1ffcc5/6798bdc2-7ad4-4907-b39d-ddfc460e73cd/image.png)
    
    3.4.    Compose your Supabase project info and add into your Trickle project knowledge. Supabase project url and api key are collect from step 3.1. And the Site URL is collect from step 3.2.
    
    ```markdown
    #### Supabase
    - Supabase project url: `https://XXXX.supabase.co`
    - Supabase anon public api key: `XXXX`
    - Site URL for Authentication: `https://XXXX.trickle.host`
    - Redirect URL for post authentication: `https://XXXX.trickle.host/onauthsuccess`
    - Requirements when integrating with Supabase:
    	- For authentication, remember to build page (served at route /onauthsuccess) to handle the process after the user finish sign up and the email verification.
    	- For database, always use SQL file for table configuration, including add and update operations. Each time database table configuration is needed, create an SQL file in the directory `supabase_operations/`. The naming format for the SQL files is as follows: `supabase_operations/1_create_chat_table.sql`. The leading number is very important, it will increase by one for a new SQL file each time the database table configuration is need to updated. For each `create` command used in SQL file, always add `check if exist` logic for it. And remember to info the user to copy the sql file content and run it in the supabase project.
    	- Use `https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2` to import the supabase javascript client.
    	- Use the following way to create supabase client.
    	```
    	const supabase = window.supabase.createClient(
    	    '<Supabase project url>',
    	    '<Supabase anon public api key>',
    	    '<options>'
    	);
    	```
    ```

4. Add account sign up and sign in using Supabase authentication.
    
    4.1.    By using *“Add login/signup page using Supabase.”* to our AI, our AI will automatically build the onboarding page, and integrate with Supabase using authentication api.
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/06fc6059-5ec4-43ed-bee7-1d888c1ffcc5/5d510d79-88c0-450d-a085-c9701f555087/image.png)
    
    4.2.   Create an account using your app, when sign up with your email, the Supabase will send you an email to verify your email address.
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/06fc6059-5ec4-43ed-bee7-1d888c1ffcc5/545f675d-2c91-4778-85f5-ddf09d104088/image.png)
    
    4.3.    Check the user account from your Supabase project.
    
    ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/06fc6059-5ec4-43ed-bee7-1d888c1ffcc5/07d09cc1-85d9-4c6b-94c7-3a5795fef838/image.png)
    
5. Save/Fetch data using Supabase database.
    
    By using prompt like *“Save data using Supabase database.”,* our AI agent will automatically the data struct and using Supabase database api to save/fetch data.
    
    Below is an example:
    
    5.1.    *“All the login user can submit question or answer other user question, and save the question using Supabase database.”*

    5.2.    Copy and paste the sql file content to your Supabase project and run the sql script.
